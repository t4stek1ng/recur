There is a multiple-choice question: Consider an electron in the n = 4, l = 1 state in hydrogen. Which of the following final states can NOT be reached by an allowed transition?. The options are: A. n = 3, l = 2, B. n = 3, l = 1, C. n = 3, l = 0, D. n = 2, l = 0. Why the correct answer is: D. n = 2, l = 0?

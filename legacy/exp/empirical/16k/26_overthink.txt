Given a single-tape deterministic Turing machine \( M \) with tape alphabet \(\{\text{blank}, 0, 1\}\), and a positive integer \( n \), we need to determine which of the following problems are decidable:

I. The computation \( C \) lasts for at least \( n \) steps.

II. The computation \( C \) lasts for at least \( n \) steps, and \( M \) prints a 1 at some point after the \( n \)-th step.

III. \( M \) scans at least \( n \) distinct tape squares during the computation \( C \).

### Analysis:

1. **Problem I:**
   - The computation \( C \) can either terminate (halting machine) or run indefinitely (non-halting machine).
   - For a halting machine, we can simulate \( M \) for \( n \) steps. If it hasn't halted, it will last at least \( n \) steps. If it has halted, we check if the halting time \( k \) is at least \( n \).
   - For a non-halting machine, the computation will last indefinitely, so it will definitely last at least \( n \) steps.
   - Therefore, Problem I is decidable.

2. **Problem II:**
   - We need to check if \( M \) prints a 1 after the \( n \)-th step.
   - For a halting machine, this can be determined by simulation. For a non-halting machine, we can't be sure because the machine might print 1s infinitely or not.
   - Therefore, Problem II is not decidable.

I. The computation \( C \) lasts for at least \( n \) steps.
II. The computation \( C \) lasts for at least \( n \) steps, and \( M \) prints a 1 at some point after the \( n \)-th step.
III. \( M \) scans at least \( n \) distinct tape squares during the computation \( C \).

### Analysis:

1. **Problem I:**
   - If \( M \) is halting, the computation will terminate after a finite number of steps. We can simulate \( M \) to check if it takes at least \( n \) steps.
   - If \( M \) is non-halting, the computation will last indefinitely, which means it will certainly last more than \( n \) steps.
   - Therefore, problem I is decidable.

2. **Problem II:**
   - If \( M \) is halting, we can simulate it to check if it prints a 1 after the \( n \)-th step.
   - If \( M \) is non-halting, it might loop indefinitely without printing a 1 after the \( n \)-th step.
   - Therefore, problem II is undecidable.

3. **Problem III:**
   - If \( M \) is halting, we can simulate it to count the number of distinct tape squares it scans.
   - If \( M \) is non-halting, it might scan a finite number of squares, possibly less than \( n \).
   - Therefore, problem III is undecidable.

- Problem I is decidable.
- Problems II and III are undecidable.
